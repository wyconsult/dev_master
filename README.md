# LicitaTraker

Sistema de gestão de licitações com integração à API ConLicitação.

## 🚀 Execução Local

```bash
npm install
npm run dev
```

Acesse em: `http://localhost:5000`

## 📋 Funcionalidades

- Dashboard com estatísticas reais
- Boletins diários (manhã, tarde, noite)
- Licitações com filtros avançados  
- Sistema de favoritos
- Autenticação de usuários
